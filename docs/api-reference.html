中亚胡杨林生态系统保护数据共享平台后端接口文档。

认证方式

本平台使用 Django Session 认证机制：

  1. GET /api/auth/csrf/ 获取 CSRF Token
  2. POST /api/auth/login/ 登录建立 Session
  3. 后续请求携带 sessionid Cookie 和 X-CSRFToken 请求头

权限体系

  • 功能权限：11 项细粒度功能权限，通过用户组分配
  • 数据访问权限：基于 access_groups 的数据隔离

错误处理

所有错误响应包含 detail 字段描述错误原因。
